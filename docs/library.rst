.. _library:
**************************
Custom XRP Library Reference
**************************

In this chapter, we give full list of all commands provided by custom XRP  MicroPython
library. We assume that the user has already installed this library on the
robot, as described in the :ref:`Quickstart Guide <quickstart>`.


This document describes version 1.0 of the library. It is intended to be used
with MicroPython 1.20 or later.
