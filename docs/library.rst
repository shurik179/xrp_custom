.. _library:
**************************
Custom XRP Library Reference
**************************

In this chapter, we give full a list of commands provided by  XRP  MicroPython
library. We assume that the user has already installed this library on the
robot, as described in the :ref:`Quickstart Guide <quickstart>`.


This document describes version 1.0 of the library. It is intended to be used
with MicroPython 1.20 or later.

This lists includes :

* all commands that are part of the custom library (these are the commands related to display and reflectance array) 

* most commonly used commands from the default `XRPLib` library; see 
  `API reference <https://open-stem.github.io/XRP_MicroPython/api.html>`__ for full list.
