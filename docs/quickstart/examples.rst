More examples
=============
Now that you have learned how to write and save programs to the robot,
it is time to explore XRP capabilities. To help with that, we have provided a
number of examples, which can be found in `python/examples` folder of the github directory
you had downloaded previously. Try opening and running them to see what the robot can do.

 Below is the list of provided examples (as of May  1, 2026):

* `self-test.py` - comprehensive self-test of all robot functions; it is recommended to run this test first 
  to make sure that everything is working properly. However, it is long, so it is easier to use one of the 
  examples below as satrting point for writing your own programs. 
* `basic_test.py` - basic test of robot operation, including OLED display, LEDs, and buttons
* `drive_test.py` - testing high-level drive commands, such as
  `go forward for 10cm` or `turn 90 degrees`
* `linearray_test.py` - testing reflectance sensor array
* `line_following.py` - folowing the line. 

All of these examples are amply commented, so it should be easy to understand
how the  code  works and how to modify it.
